# AI-Powered Smart Dashcam & Road Safety System

## Overview

This project is an AI-powered Smart Dashcam and Road Safety System developed to improve driver safety and reduce road accidents using real-time computer vision and intelligent monitoring.

The system uses:

* **Driver View Camera** → Monitors driver behavior
* **Road View Camera** → Monitors road conditions and traffic risks

The project provides real-time alerts, accident prevention assistance, and emergency support using AI, OpenCV, and Raspberry Pi integration.

---

# Key Features

## Driver Monitoring System

The Driver View Camera continuously analyzes the driver's condition and behavior.

### Detects:

* Driver drowsiness
* Eye closure
* Yawning
* Driver distraction
* Head movement abnormalities
* Unsafe driving behavior

### Alerts:

* Voice alerts
* Visual warnings
* Real-time notifications

---

## Road Monitoring System

The Road View Camera analyzes the road environment in real time.

### Detects:

* Lane departure
* Obstacles on road
* Vehicle proximity
* Collision risks
* Traffic conditions
* Unsafe road situations

### Safety Actions:

* Driver warning alerts
* Risk indication system
* Smart accident prevention assistance

---

# SOS Emergency System

The project includes an SOS emergency support feature.

### Features:

* Emergency trigger support
* Accident assistance
* Real-time emergency alerts
* Quick response safety mechanism
